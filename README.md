# Simple Web Project

A simple HTML, CSS, JavaScript, and Node.js project.

## Project Structure

```
render_1/
├── public/
│   ├── index.html
│   ├── styles.css
│   └── script.js
├── server.js
├── package.json
└── README.md
```

## Installation

1. Install dependencies:
```bash
npm install
```

## Running the Project

Start the server:
```bash
npm start
```

Then open your browser and navigate to `http://localhost:3000`

## Features

- Express.js server serving static files
- Interactive button with click counter
- Modern, responsive design
- Gradient styling
